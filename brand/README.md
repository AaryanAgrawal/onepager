# Brand Assets

Example brand assets for the one-pager generator. Replace these with your own company's branding.

## Files

| File | Purpose |
|------|---------|
| `logo-500x500.png` | Square symbol logo |
| `logo-w-type-light.png` | Full logo with text (light background) |
| `logo-w-type-dark.png` | Full logo with text (dark background) |
| `favicon.svg` | Browser tab icon |
| `colors.css` | CSS custom properties for brand colors |
| `logo-250-base64.txt` | Base64-encoded logo for embedding in templates |

## How to Use Your Own Brand

1. Replace the files in this directory with your own assets (keep the same filenames)
2. Update `colors.css` with your brand colors
3. Update the `.env.local` file with your company name, email, phone, website
4. The app will use these as defaults when generating documents
