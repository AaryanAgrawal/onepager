# References

Drop files here that Claude Code should use as style references when creating one-pagers.

## What to put here

- **HTML files** — past one-pagers or examples you like. Claude will match the style.
- **Images** (PNG, JPG) — screenshots of designs for layout inspiration.
- **URLs** — add to `urls.txt` (one per line). Claude can fetch and study them.

## How it works

When you ask Claude Code to create a new one-pager, it reads this folder first and uses whatever it finds as style guidance. The more examples you provide, the better it matches your preferred look.
